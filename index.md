As an incoming Ph.D. student, I have already seen a tons of lectures. 
I know firsthand that finding the **right courses** and their materials, even **videos**, can be very time-consuming. 
Although cutting-edge LLM can serve as an assistant to answer questions and solve problems, we usually prefer learning by participating in classes, whether online or offline. 
Therefore, I created this awesome-course-collection project to simply record the courses I have collected. 
It’s not only for my own reference, but perhaps anyone who comes across this project can also benefit from it.

I will try to keep this project updated. If you know of any suitable courses, feel free to open a GitHub issue.
